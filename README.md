# 咫尺域 - 工作伪装工具（最终版）
一个基于 Tauri v2 的桌面应用程序，提供人脸检测和快速伪装功能，帮助你在工作中保持专注。

## 主要功能
- **人脸检测监控**：使用摄像头实时检测人脸，当检测到多张人脸时自动跳转到伪装页面
- **全局快捷键**：`Ctrl+Shift+F` 快速切换到伪装页面
- **智能会话恢复**：从伪装页面返回后自动登录，摸鱼时间持续累计
- **统计数据**：记录本次会话时长、累计摸鱼时间和人脸检测次数
- **用户认证**：支持登录注册功能，数据本地存储
- **轻量级设计**：精简的界面，专注核心功能

## 技术栈
- **前端**：HTML/CSS/JavaScript + Vite
- **后端**：Rust + Tauri v2
- **数据库**：SQLite（rusqlite）
- **人脸检测**：浏览器 FaceDetector API（优先）或 Canvas 亮度/运动分析（fallback）

## 项目概况
这是一个用于工作场景的桌面伪装工具，通过人脸检测来触发伪装页面跳转，帮助用户在被他人看到时快速切换到"工作"界面。项目采用前后端分离架构，前端负责UI和人脸检测逻辑，后端处理数据存储。整体代码量小，易于理解和扩展。

### 人脸检测实现思路
1. **优先使用 FaceDetector API**：现代浏览器（Chrome/Edge）支持的原生API，直接检测人脸数量和位置。
2. **Fallback 方案**：不支持时，使用 Canvas 分析图像亮度和帧间差异来模拟检测。
3. **触发条件**：支持API时检测到≥2张人脸触发跳转；Fallback时连续3次检测到亮度>70或运动>12%触发。
4. **防误触发**：启动后延迟2秒再开始检测，使用`autoOpenTriggered`标志防止重复触发。

### 会话持久化设计
- 登录状态：用户名和登录时间戳保存在`localStorage`，跨页面跳转不丢失
- 返回恢复：从伪装页面返回时，自动读取保存的状态并登录
- 时间累计：返回时自动计算离开期间的时长并存入数据库，顶部计时器重置为0，底部累计时间持续累加

```bash
npm install
```

### 开发模式
```bash

```

### 构建应用
```bash
npm run tauri build
```

## 使用说明
1. 启动应用后，进行登录或注册
2. 登录后，右上角计时器开始计时，底部显示累计摸鱼时间
3. 点击"开启人脸监控"按钮，允许摄像头权限
4. 等待2秒后，监控正式开始
5. 当检测到多张人脸时，应用会自动在当前窗口跳转到伪装页面
6. 使用`Ctrl+Shift+F`快捷键可手动切换到伪装页面
7. 从伪装页面按`Alt+←`返回，应用自动恢复登录状态

## 功能说明
### 人脸检测
- 支持原生 FaceDetector API 和 Canvas fallback 实现
- 启动延迟2秒，避免初始化误判
- Fallback 阈值：亮度>70 或 帧差>12%，需连续3次
- 实时显示检测状态和人脸数量

### 伪装页面
当前配置为 CNKI 知网（https://www.cnki.net/），在当前窗口直接跳转，不使用新窗口。可在`main.js`的`openDisguisePage()`函数中修改目标URL。

### 计时与统计
- 顶部计时器：显示本次会话的实时时长（HH:MM:SS）
- 今日摸鱼时间：显示数据库中累计的摸鱼分钟数
- 人脸检测次数：累计检测到人脸的次数
- 摸鱼时间每分钟自动存入SQLite数据库

### 数据存储
- 用户账号：`localStorage`存储（disguise_tool_users）
- 会话状态：`localStorage`存储（disguise_tool_session、disguise_tool_login_time）
- 统计数据：SQLite数据库（disguise_tool.db），位于用户应用目录

### 打包说明
基于 Tauri v2，可打包成独立的可执行文件，Windows输出`.msi`安装包，打包后无需额外依赖即可运行。

## 注意事项
- 首次运行需要摄像头权限
- 人脸检测在低光环境下可能不准确
- 伪装页面URL可根据实际需求在`main.js`中修改
- 数据仅本地存储，不涉及网络传输
- 从伪装页面返回时，请使用浏览器后退功能（`Alt+←`）
